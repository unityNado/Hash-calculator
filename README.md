RU:
Краткое руководство пользователя

1. Выбор режима работы
В верхней части окна с помощью переключателей выберите, что вы хотите хэшировать:
Текст — для обработки текстовой строки.
Файл — для вычисления контрольной суммы файла.

2. Ввод данных
Если выбран режим "Текст": введите или вставьте нужный текст в поле "Введите текст:".
Если выбран режим "Файл": нажмите кнопку "Обзор...", чтобы выбрать файл на вашем компьютере. Имя выбранного файла отобразится рядом с кнопкой.

3. Выбор алгоритма
Выберите нужный алгоритм хэширования (MD5, SHA-1 или SHA-256). По умолчанию выбран MD5.

4. Получение результата
Нажмите кнопку "РАССЧИТАТЬ". Хэш-сумма появится в поле "Результат:" внизу окна.

5. Копирование
Чтобы скопировать полученный хэш, нажмите кнопку "КОПИРОВАТЬ". Текст автоматически сохранится в буфер обмена, и вы сможете вставить его в любой документ или мессенджер (Ctrl+V).



EN:
Brief User Guide

1.Select Mode
At the top of the window, use the radio buttons to select what you want to hash:
Text — to process a text string.
File — to calculate the checksum of a file.

2. Input Data
If "Text" mode is selected: Enter or paste the desired text into the "Enter text:" field.
If "File" mode is selected: Click the "Browse..." button to select a file on your computer. The name of the selected file will appear next to the button.

3. Select Algorithm
Choose the desired hashing algorithm (MD5, SHA-1, or SHA-256). MD5 is selected by default.

4. Get Result
Click the "CALCULATE" button. The hash sum will appear in the "Result:" field at the bottom of the window.

5. Copying
To copy the resulting hash, click the "COPY" button. The text will be automatically saved to the clipboard, and you can paste it into any document or messenger (Ctrl+V).
